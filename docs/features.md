A blog + ecommerce system
Features:
-Admin users can post blog posts
-Admin users can add categories and products
-Slug will be used in post and product and categories urls to achieve better on-page SEO
-Product has images, description and price
-Normal users can like and comment on blog posts
-Normal users can purchase products
-Normal users will recieve an email and a text message on each purchase
-We need to have RSS and Sitempap
-Will be deployed using Docker
-Will use zarinpal as the paymant system
-promotion code usage
-Git usage
-CI/CD pipeline using github actions
-use CDN for static content
-production level deployment
-Normal users can use API to check products, prices, blog posts
